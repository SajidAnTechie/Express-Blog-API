## Project Setup

```bash

npm install

npm install -g nodemon

```

## Configuration Setup

```env
PORT=5000
NODE_ENV=development

SECRET_KEY=your secreat key


MONGO_URL=MongoDb url

SMTP_HOST=Your smtp host
SMTP_PORT=port
SMTP_EMAIL=username
SMTP_PASSWORD=password
FROM_EMAIL=your email
FROM_NAME=your name

```

## Run the server

```bash
npm run dev
```
